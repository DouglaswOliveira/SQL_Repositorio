## INSTRUÇÃO SELECT :

**Consulta básica :**
- Mostre-me Todos os Dados da tabela chamada MY_TABLE
  - Codigo :

 ```javascript
SELECT * FROM MY_TABLE;
```

**Consulta Com Filtro e Classificação :**

 ```javascript
SELECT *
FROM MY_TABLE
WHERE COLUMN1 > 100
ORDER BY COLUMN2;
```
